# ПайсМобайл Flask

Запуск на macOS: дважды нажмите `start.command`.

В этой версии файл запуска больше не переустанавливает Flask каждый раз. Он сначала ищет уже установленный Flask в `venv`, `.venv`, соседних папках и системном `python3`. Если Flask не найден, установка выполняется только один раз.
